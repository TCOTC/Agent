思源 Agent 插件 — 开发者学习站

本站面向「希望把 AI Agent 集成进思源笔记」的开发者，用中文系统讲解大语言模型与 Agent 的基本概念、常见工程范式， 并对本仓库 demo 插件的目录结构、运行循环、工具层与思源内核交互方式做对照说明。阅读时可配合本地 src/ 源码一起翻看。

如何使用本站：按左侧顺序从「基础」读到「工具」即可形成完整心智模型；若你更关心「这份 demo 在干什么」， 可直接跳到「本插件架构」与「源码导览」。

本 demo 插件一句话概括

在思源进程内，用浏览器 fetch 调用「OpenAI 兼容」的 chat/completions 接口， 把模型返回的 tool_calls 交给本地 TypeScript 执行；执行层通过 fetchSyncPost 访问思源内核 REST API，且所有写操作与可选 SQL 都经用户弹窗确认；可访问的文档范围由用户显式维护的「工作集」限定。
